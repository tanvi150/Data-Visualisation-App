# Data Visualisation App Extensions
## Introduction to Programming 2 (2025)

This project extends a Data Visualisation App by adding interactive visualization components and menus, developed using p5.js. The app allows users to explore various types of visualizations dynamically, using custom-built tools and in-built templates.

## Features
### Interactive Visualization Components
- Animated Globe: Displays global point population data interactively.
- Graph Generator: Supports five chart types with a custom CSV parser for dynamic data input.
- Interactive Heatmap: Provides real-time feedback with dynamic visual changes.

### Menus and Extensions
- 5 menus were already in-built, offering a wide range of visualization options.
- Developed 3 additional menus to extend the app's functionality.

### In-Built Templates
- Interactive Pie Chart
- Double Way Bar Graph
- Scatterplot
- Two Line Graphs

### Technologies Used
- p5.js: core library for graphics and interactivity
- HTML/CSS: User interface elements
